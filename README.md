X86架构是芯片巨头Intel设计制造的一种微处理器体系结构的统称，我们所用的PC绝大部分都是X86架构

x86采用CISC（Complex Instruction Set Computer，复杂指令集计算机）架构。

与采用RISC不同的CISC处理器程序的各条指令是按顺序串行执行的，每条指令中的各个操作也是按顺序串行执行的。顺序执行的优点是控制简单，但计算机各部分的利用率不高，执行速度慢。
